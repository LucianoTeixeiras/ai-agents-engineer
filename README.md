## 📚 Conteúdo da Formação (Trilha Principal)

Para aproveitar a formação ao máximo, siga esta ordem sequencial. Alunos com conhecimento prévio podem optar por avançar módulos introdutórios.

   Aprenda a linguagem de programação mais famosa do momento! Este é o curso certo para quem quer dar seus primeiros passos em Python.
8. **[Criando seu Setup para Programação Python](./cursos-principais/08-setup-python)** (2h 41min) - *Por Juliano Faccioni*  
   Instale Python sem mistérios! Este curso te ensina a navegar no terminal e configurar as principais IDEs da forma certa.
9. **[Engenharia de Prompts](./cursos-principais/09-engenharia-de-prompts)** (2h 9min) - *Por Adriano Soares*  
   Domine a arte de criar prompts eficazes para modelos de linguagem como o ChatGPT!
